# 教授大戰 (Professor Wars) 🎓🔥

這是一個使用 **VB.NET** 開發的 2D 彈道射擊遊戲

## 📝 遊戲簡介
在學術界，除了論文，有時候我們需要用「實力」說話！這是一款雙人對戰遊戲，兩位教授將在校園地圖中，利用教授的特有武器來擊敗對手。

## 🎮 遊戲特色
* **經典彈道玩法**：計算角度與力度，精準打擊。
* **特色角色**：可選擇不同的教授，擁有獨特的投擲物。

## 🚀 如何執行
1.  確保電腦已安裝 [.NET Framework](https://dotnet.microsoft.com/download)。
2.  下載此專案並解壓縮。
3.  使用 Visual Studio 打開 `ProfessorWars.sln`。
4.  按下 `F5` 開始編譯並執行。
